# 面试题库

技术面试经常从一个简短问题开始：“什么是事务？”“缓存击穿和缓存穿透有什么区别？”“接口超时后能不能重试？”

面试题库帮助读者把已经理解的知识组织成简洁、准确的口头回答。既包含基础概念和机制，也包含并发、故障与系统设计等场景问题。

## 怎样使用题库

看到问题后，可以先遮住答案，尝试在三十秒到一分钟内回答，再检查自己是否说清了：

```text
它是什么
→ 为什么需要它
→ 关键机制是什么
→ 在什么边界下会失效或需要取舍
```

定义题应先说准确，场景题应先给出判断和原因。遇到不能解释的机制或工程取舍时，可以沿题目中的链接回到知识正文。
